# Schmetterling-ZIP

Ein JavaFX-Tool zum Erstellen und Entpacken von Polyglot-Archiven.
(JavaFX utility for creating and extracting polyglot archives).

## Features
* Verpacken von Dateien in ein kombiniertes Format (.myash).
* Unterstützung von versteckten Sektoren über benutzerdefinierte Byte-Marker.
* Getrennte Extraktion von öffentlichen und geheimen Daten.

## Anforderungen (Requirements)
* Java 21
* Gradle
